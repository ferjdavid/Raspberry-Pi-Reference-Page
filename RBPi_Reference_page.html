Raspberry Pi

What is Raspberry Pi?

Raspberry Pi is a low cost, credit card sized computer.
It is a fully featured, internet connected Linux
computer that is easy to embed into prototypes.

Extensions

Raspberry Pi is also a platform for interactive devices
and the Internet of Things.

GPIO pins and HAT extensions make it easy to add sensors,
physical controls, outputs and interfaces to your projects.

Comparing Raspberry Pi & Arduino

Good for processing data / Linux OS / HDMI / Audio
output / USB / Wifi / Bluetooth / Only digital GPIO pins /
High power consumption : 750mW+.

Choose Raspberry Pi for handling video or graphic interfaces /
Projects that process lots of data ( ie. data visualization ) /
Computer vision and other performance-dependant processes.

Good for controlling inputs & outputs / Only runs the
code you put on it / No OS or firmware / Digital and
Analog GPIO pins / Low power consumption : 175mW.

Choose Arduino for controlling lots of sensors, small LCDs and motors /
Low power applications (ie. battery powered projects) /
Analog sensors.

Setting-up Raspberry Pi

A few things to know first.

Label your Pi and accessories.

Everyone's Pi and power supply look the same.
Labeling will help me get it back to you if you forget something.

Attach USB & power last.

Just like with Arduino, we want to make sure that our circuits
are complete before powering. With Raspberry Pi, this also ensures
all peripherals initialize properly while it boots up. Make sure
your Pi is off while making changes to attached circuits.

Python

Let's get some code running on our Pis.

You're already familiar with the Arduino IDE

The arduino programming language is based on C, and
used the as the environment to program and run code.

Thonny is similar for Python

We will be using the Python programming language in the
Thonny IDE. It is very light on syntax ( easy for us to read and write )
and interfaces well with RPi peripherals like SenseHAT.

Comparing Python to Arduino & Processing

Interpreted, generally runs slower / Typically less lines of code, and more legible / Indentation is part of the
code, incorrect spacing will cause errors / Dynamically
typed variables, decided at run time / Not tied to a
specific IDE (VS Code is a good one) / No built in looping.

Compiled, generally runs faster / More syntax /
Indentation is purely visual, but a good
practice / Statically typed variables (ie. int) / Bundled
with an IDE / Have a built in
looping function: loop( ) or draw( ).

Indentation and Dynamic typing

Indentation

In Python, indentation is part of the code, replacing brackets.
This means it will cause errors if incorrect. You can use either spaces
or tabs, but it's best to pick one.

Dynamic Typing

Variables don't need to be set to a specific type of data,
and can change at runtime.

Sense HAT

Sense HAT is an add-on board including a grid of leds,
and sensors including: Gyroscope / Accelerometer / Magnetometer / Temperature / Barometric pressure /
Humidity / Joystick & physical input.
